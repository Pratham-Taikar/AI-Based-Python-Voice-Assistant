# 🤖 AI Desktop Assistant

A powerful, voice-controlled desktop assistant with AI capabilities built with Python.

## ✨ Features

- 🎤 **Voice Recognition** - Control with your voice
- 💬 **AI Conversations** - Powered by Groq GPT
- 📁 **File Management** - Create files and folders
- 🖥️ **App Control** - Open applications
- 🔍 **Web Search** - Search Google instantly
- ⏰ **Reminders** - Set timed reminders
- 🎯 **Focus Mode** - Pomodoro timer
- 📸 **Screenshot Analysis** - OCR text extraction
- ⚡ **Voice Shortcuts** - Custom command sequences
- 🌙 **Modern GUI** - Beautiful dark mode interface

## 📦 Installation

### For Users (Easy)
1. Download `AI_Assistant_Setup.exe`
2. Run the installer
3. Launch from desktop

### For Developers
```bash
# Clone repository
git clone https://github.com/yourusername/AI_Desktop_Assistant
cd AI_Desktop_Assistant

# Install dependencies
pip install -r requirements.txt

# Setup API keys
# Create .env file and add your Groq API key

# Run
python gui_assistant.py
```

## 🚀 Quick Start

1. **Launch** the application
2. **Click "Listen"** or type commands
3. **Try these commands:**
   - "What time is it?"
   - "Create file named notes"
   - "Open calculator"
   - "Focus for 25 minutes"
   - "Remind me in 10 minutes"

## 💻 System Requirements

- **OS:** Windows 10/11, macOS 10.14+, Linux
- **RAM:** 4GB minimum (8GB recommended)
- **Storage:** 500MB
- **Microphone:** Required for voice commands
- **Internet:** Required for AI features

## 🛠️ Tech Stack

- **Language:** Python 3.11
- **Speech Recognition:** SpeechRecognition, pyttsx3
- **GUI:** CustomTkinter
- **AI:** Groq API
- **OCR:** PyTesseract
- **Other:** psutil, Pillow, python-dotenv
